    • grid-template-columns: Permite crear las columnas para organizar el contenido, debemos indicar mediante tamaños o el valor auto la cantidad.
    • grid-template-rows: Permite crear y asignar el tamaño de las filas, a diferencia de columns NO define la cantidad maxima de las mismas.
    • gap: Permite definir el epaciado entre celdas, aplicando a filas y columnas respectivamente, aunque tambien exiten los atributos row-gap y column-gap.
